# Code of Conduct

This project adopts the
[Contributor Covenant v2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/).

Report violations privately via GitHub Security Advisories on the project
repository, or by opening a confidential issue requesting maintainer contact.
All reports are handled confidentially. Enforcement follows the Covenant's
standard ladder (correction → warning → temporary ban → permanent ban).
